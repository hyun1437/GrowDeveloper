# GrowDeveloper
개발자키우기
턴제 RPG 게임 제작하기
  
  
## 1. 기준
### 코드 컨벤션
- 클래스 명 : 카멜 케이스
- 함수 및 변수명 : 스네이크 케이스
- 사용 툴 : Python
  
  
## 2. 게임소개
### 스토리
개발자가 되기 위한 여정! 개발자 키우기에서 현생에서 못 다 이룬 개발자 취업의 꿈을 이뤄보세요!
잠에서 깨어나보니 당신은 개발자 취준생이 되었따!? 개발자 예비지망생 -> 진짜 개발자로 전직!
내배캠마을에 입장하였습니다.
